# mike.fi

My home page at mike.fi, hosted with GitHub Pages using Jekyll.

## Local setup

Installing from scratch with homebrew:

```sh
brew install rbenv
rbenv install 2.5.3
rbenv shell 2.5.3
gem install bundle
rm Gemfile.lock
bundle install
```

Consequently:

`bundle exec jekyll serve`
